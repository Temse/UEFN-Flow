# UEFN Flow

Welcome to **UEFN Flow**! This is a modern, high-performance web application designed to help you manage your projects effortlessly.

**Live Demo:** [https://temse.github.io/UEFN-Flow/#/](https://temse.github.io/UEFN-Flow/#/)

## Features
- **Project Management:** Create, manage, and organize your work.
- **Customizable Interface:** Choose from various color themes, dark/light modes, and font sizes.
- **Visual Effects:** Dynamic background glows and glassmorphism UI elements (can be disabled in Performance Mode).
- **Responsive Design:** Optimized for both desktop and mobile views.
- **Localization:** Supports multiple languages out of the box.

## Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
4. Open the browser to view the application

## Settings & Preferences

You can access the Preferences panel directly in the app to configure:
- **Display & UI:** Dark/Light mode and font sizes.
- **Brand Colors:** Customize primary and secondary accent colors.
- **Effects & Visuals:** Toggle UI animations, glassmorphism, and the interactive Mouse Glow Effect.
- **Performance Mode:** Easily switch off heavy visual effects for a smoother experience on older devices.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion (for animations)
- Lucide React (for icons)
